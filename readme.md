# SJA 分析器 v3.0 开源仓库

## SJA 简介

SJA 分析器，原名 Scratch Json Analyser。是一个具有**作品结构分析**、**作品相似度对比**和**作品跨站迁移（仍在筹划制作中）** 功能的 Scratch 文件工具。

SJA 原版采用 Scratch 实现，旧版 SJA 网站采用 Python 后端实现。本仓库为最新版本 SJA(v3)的源代码仓库，采用 JS 实现。
